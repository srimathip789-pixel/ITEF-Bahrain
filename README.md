# LMS Tutor Application

This is a React + TypeScript application built with Vite.

## Project Setup

1.  **Install Dependencies** (if you haven't already):
    ```bash
    npm install
    ```

## Running the App

### Development Server
To start the local development server:
```bash
npm run dev
```
Then open [http://localhost:5173](http://localhost:5173) in your browser.

### Production Build
To build the application for production:
```bash
npm run build
```
The build artifacts will be stored in the `dist/` directory.

## Project Structure
- `src/components`: Reusable UI components (Header, etc.)
- `src/pages`: Application pages (Login, Dashboard, CourseDetails)
- `src/App.tsx`: Main application component and routing configuration
